🚀 HOW TO PUBLISH TO WEB (5 MINUTES):

STEP 1: Go to GitHub Settings
   → https://github.com/abremers/biogradix/settings/pages

STEP 2: Configure GitHub Pages
   → Source: "Deploy from a branch"
   → Branch: "main"
   → Folder: "/ (root)"
   → Click Save

STEP 3: Wait 1-2 Minutes
   → GitHub will show blue "Building..." banner
   → Wait for green checkmark
   → It will show: "Your site is published at https://biogradix.com"

STEP 4: Verify
   → Open https://biogradix.com/
   → You should see the homepage
   → Click EN button to see English version
   → Click footer links to verify they work
